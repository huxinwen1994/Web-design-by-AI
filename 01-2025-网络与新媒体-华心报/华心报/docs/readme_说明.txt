华心报新闻网站 - 项目说明文档
1. 项目名称与简介
项目名称：华心报 - 湖湘文化新闻网站

简介：华心报是一个基于HTML、CSS和JavaScript技术栈开发的专注于湖湘文化传播的新闻网站，涵盖民俗、文学、方言、饮食、工艺五大领域，通过现代中式设计与丰富的交互功能，并集成了搜索筛选、内容展开折叠、表单验证等多种交互功能，为用户提供便捷的阅读体验。

2. 目标用户与应用场景
目标用户：
文化爱好者：对湖湘民俗、历史、文学感兴趣的人群。
本地居民：关注湖南方言、饮食、工艺的本地用户。
游客与研究者：计划或正在研究湖湘文化的游客、学者。
应用场景：
文化探索：用户通过“观民俗”“赏工艺”等板块了解湖南传统文化。
学习交流：在“品文学”页面参与文学作品讨论或报名参赛。
生活指南：通过“尝味道”页面查找湘菜食谱或饮食习俗。

4. 主要交互功能介绍
功能1: 新闻搜索/筛选功能
- 位置：首页搜索框
- 实现：原生JavaScript
- 功能描述：
  - 用户可以在搜索框输入关键词
  - 点击"搜索"按钮或按下回车键触发搜索
  - 系统会筛选出标题或内容包含关键词的新闻卡片
  - 不匹配的卡片会被隐藏
  - 显示搜索结果数量提示

功能2: 内容展开/折叠功能
- 位置：所有新闻卡片
- 实现：原生JavaScript
- 功能描述：
  - 新闻卡片初始状态只显示首段内容
  - 点击"展开更多"按钮可查看完整内容
  - 点击"收起内容"按钮折叠为首段显示
  - 提升页面整洁度和阅读体验

功能3: 移动端响应式菜单
- 位置：导航栏
- 实现：原生JavaScript + CSS
- 功能描述：
  - 在移动设备上显示汉堡菜单按钮
  - 点击按钮展开/收起导航菜单
  - 适配不同屏幕尺寸

功能4: 返回顶部按钮
- 位置：页面右下角
- 实现：原生JavaScript
- 功能描述：
  - 页面滚动超过300px时显示按钮
  - 点击按钮平滑滚动回页面顶部
  - 悬停时颜色变化提供视觉反馈

功能5: 表单验证功能
- 位置：登录和注册页面
- 实现：原生JavaScript
- 功能描述：
  - 验证必填字段不为空
  - 验证邮箱格式正确性
  - 验证密码长度（至少6位）
  - 验证两次输入密码一致性
  - 显示错误提示信息
  - 提交成功后显示成功消息

5. Prompt 使用情况简述

本项目在开发过程中充分利用了AI大模型辅助编码的能力：
初始阶段
- 使用Prompt描述网站整体架构和需求
- 生成基础的HTML页面结构和导航系统
- 生成CSS样式框架和布局系统

迭代阶段
- 通过追加Prompt添加搜索功能
- 优化响应式设计，适配移动设备
- 添加交互功能（展开折叠、表单验证等）
- 完善样式细节和用户体验

技术实现要点
- 所有JavaScript功能均使用原生实现，无依赖外部库
- CSS采用Flexbox和Grid布局技术
- 响应式设计支持桌面、平板和手机三种尺寸
- 代码结构清晰，注释完整

详细的Prompt内容请参见 `prompt_说明.txt` 文件。

6. 网站运行与测试说明

运行环境
- 浏览器：Chrome、Firefox、Safari、Edge等现代浏览器
- 无需服务器环境，直接打开HTML文件即可运行

运行步骤
1. 下载并解压项目文件
2. 使用浏览器打开 `index.html` 文件
3. 通过导航栏访问不同页面

 测试要点
 功能测试
1. 搜索功能测试
   - 在首页搜索框输入关键词（如"改革"）
   - 验证搜索结果是否正确筛选
   - 测试空关键词显示所有内容

2. 展开折叠测试
   - 查找有多段内容的新闻卡片
   - 点击"展开更多"按钮
   - 验证内容完整显示和按钮文本变化

3. 表单验证测试
   - 访问登录/注册页面
   - 测试空字段提交
   - 测试邮箱格式验证
   - 测试密码长度和一致性验证

4. 返回顶部测试
   - 向下滚动页面
   - 点击右下角返回顶部按钮
   - 验证平滑滚动效果

响应式测试
1. 使用浏览器开发者工具（F12）
2. 切换到设备模拟模式
3. 测试不同屏幕尺寸：
   - 桌面（>768px）
   - 平板（480px-768px）
   - 手机（<480px）
4. 验证布局、导航和交互功能在各尺寸下的表现

浏览器兼容性测试
- Chrome（推荐）
- Firefox
- Safari
- Edge

 7. 技术栈与特色功能

技术栈
- HTML5：语义化标签，良好的文档结构
- CSS3：Flexbox、Grid布局，媒体查询响应式设计
- JavaScript ES6：原生JavaScript实现所有交互功能

特色功能
1. 完全响应式：适配所有设备尺寸
2. 原生实现：无外部依赖，代码可控
3. 用户友好：返回顶部、平滑滚动等细节优化
4. 表单验证：前端实时验证，提升用户体验
5. 搜索筛选：快速定位目标内容
