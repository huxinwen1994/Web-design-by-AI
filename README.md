## 方法二：下载后本地运行

### 1. 克隆仓库

```bash
git clone 仓库地址
```

### 2. 进入项目目录

```bash
cd 项目文件夹
```

### 3. 使用本地服务器运行

推荐使用：

* VSCode Live Server
* Python SimpleHTTPServer
* Node.js 本地服务

例如：

```bash
python -m http.server 8000
```

浏览器打开：

```text
http://localhost:8000
```
