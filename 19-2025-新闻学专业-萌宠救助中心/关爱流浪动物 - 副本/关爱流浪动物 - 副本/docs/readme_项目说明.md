# 萌宠救助中心 - 项目说明文档

## 1. 项目名称与简介

**项目名称**：萌宠救助中心 (Animal Rescue Center)

**项目简介**：
本项目是一个关爱流浪动物的信息交互类网站，旨在为流浪动物救助机构提供一个展示平台，帮助待领养动物找到温暖的家，同时为志愿者和捐赠者提供便捷的参与渠道。网站包含动物信息展示、领养流程介绍、志愿者招募、捐助支持等核心功能。

---

## 2. 目标用户与应用场景

### 目标用户
- **潜在领养人**：希望领养流浪动物的爱心人士
- **志愿者**：愿意参与救助工作的志愿者
- **捐赠者**：希望通过物资或资金支持救助工作的人群
- **社会公众**：关注流浪动物保护的社会各界人士

### 应用场景
- 浏览待领养动物信息，筛选合适的领养对象
- 了解领养流程和要求，提交领养申请
- 报名成为志愿者，参与救助工作
- 查看捐助公示，进行在线捐赠
- 了解救助故事和机构动态

---

## 3. 网站信息结构与主要板块说明

### 页面结构（共5个主要页面）

#### 3.1 首页 (index.html)
- **导航栏**：包含所有主要页面的导航链接和搜索功能
- **轮播横幅**：展示救助现场和重点活动（带自动播放和手动切换）
- **本周推荐领养**：展示3只待领养动物的卡片
- **救助故事**：展示3个救助案例
- **中心公告**：显示最新公告列表
- **快速链接**：引导用户了解领养流程和捐助方式

#### 3.2 关于我们 (about.html)
- 机构简介
- 救助理念
- 团队介绍
- 联系方式

#### 3.3 待领养页面 (adoption.html)
- **筛选栏**：按类型（猫/狗）、年龄、体型筛选动物
- **动物卡片列表**：展示所有待领养动物的基本信息
- **领养流程说明**：侧边栏展示领养步骤和提示
- **动物详情页** (adoption-detail.html)：展示单个动物的详细信息

#### 3.4 志愿者页面 (volunteer.html)
- 志愿者招募说明
- 工作内容介绍
- 报名表单

#### 3.5 捐助支持页面 (donate.html)
- 捐助方式说明
- 物资需求列表
- 捐赠公示
- 在线捐赠表单

## 4. 主要交互功能介绍

### 4.1 轮播图交互 (已实现)
- **位置**：首页横幅区域
- **功能**：自动播放、左右箭头切换、点击圆点跳转、鼠标悬停暂停
- **技术实现**：原生JavaScript，使用定时器和事件监听

### 4.2 动物筛选功能 (新增)
- **位置**：待领养页面
- **功能**：按照动物类型、年龄、体型进行筛选显示
- **技术实现**：原生JavaScript，通过data属性标记和class切换实现

### 4.3 搜索功能 (新增)
- **位置**：所有页面的导航栏
- **功能**：根据关键词搜索动物、活动、资讯
- **技术实现**：原生JavaScript，前端模糊匹配

### 4.4 表单验证与提交 (新增)
- **位置**：志愿者报名页面、捐助页面
- **功能**：表单输入验证、提交反馈
- **技术实现**：原生JavaScript表单验证

### 4.5 内容展开/折叠 (新增)
- **位置**：捐助公示、FAQ等区域
- **功能**：点击展开更多内容
- **技术实现**：原生JavaScript切换class

## 6. 网站运行与测试说明

### 6.1 运行方式
1. **直接打开**：双击 `index.html` 在浏览器中打开
2. **本地服务器**（推荐）：使用Live Server或Python HTTP服务器运行
   ```bash
   # Python 3
   python -m http.server 8000
   # 然后访问 http://localhost:8000
   ```
