# 《疯狂动物城》电影介绍 网站项目说明

## 项目简介

这是一个关于《疯狂动物城》电影介绍的网站，采用HTML、CSS和JavaScript技术开发，包含多个页面和丰富的交互功能。



## 目标用户与应用场景

**目标用户**群体为儿影迷、动画爱好者以及对《疯狂动物城》电影感兴趣的人群。
**应用场景**包括：
1.个人娱乐与自主学习方面，网站是影迷和动画爱好者的数字观影伴侣。无论是重温经典还是为新观众预热，用户都可以通过角色介绍和剧情亮点深入了解故事，并结合创作背景和主题思想，获得更深层次的观影启发。
2.对于希望寓教于乐的家庭与教育工作者而言，网站是一个现成的教学资源库。家长们可以借助网站内容引导孩子思考电影所传递的平等、勇气等积极价值观；教育工作者则可以将网站内容作为课堂素材，围绕角色和剧情设计活动，激发学生对动物科学和社会的兴趣。
3.随着《疯狂动物城》主题园区在全球迪士尼乐园的落成以及各类线下主题展览的举办，网站也能成为游客在线下游览前后的参考指南与知识延伸，帮助他们更好地沉浸于动物城的世界。
4.对于各类内容创作者，如视频UP主、自媒体博主或学术研究者，您的网站提供了一个结构清晰、信息相对集中的资料站点，可以作为他们创作影评、解读视频或进行研究时的参考资料。

## 主要交互功能介绍

### 搜索/筛选功能
- 在首页顶部提供搜索框
- 支持按关键词搜索角色、剧情、主题等内容
- 实时筛选并高亮匹配内容
- 支持清除搜索

### 表单提交功能
- 在"联系我们"页面提供完整的表单
- 前端表单验证
- 提交成功后显示提示信息
- 5秒后自动重置表单

### 内容展开/折叠功能
- 场景详情可展开/收起
- 主题列表可展开/收起
- 平滑滚动动画效果

### 主题切换功能
- 三种主题：默认主题、深色主题、彩色主题
- 主题选择保存到本地存储
- 页面刷新后自动应用保存的主题

### 数据可视化图表
- 使用Canvas API绘制柱状图
- 三种数据视图：角色数据、主题数据、场景数据
- 按钮切换不同数据维度
- 响应式设计，自适应屏幕大小

### 轮播图功能
- 自动播放
- 手动切换
- 指示点导航
- 鼠标悬停暂停

## 网站运行与测试说明

### 开发环境
- 操作系统：Windows 10
- 浏览器：Google Chrome（推荐）、Mozilla Firefox、Microsoft Edge、Apple Safari
- 编程语言：HTML、CSS、JavaScript
- 开发工具：Visual Studio Code
- 图片处理工具：canvas

### 开发说明
- CSS文件：`css/style.css`
- JavaScript文件：`js/main.js`（包含所有交互功能）
- 图片资源：`assets/` 文件夹

### 测试环境
- 浏览器：Google Chrome（最新稳定版）
- 网络环境：Wi-Fi、有线网络
- 设备：个人电脑、平板电脑、智能手机

### 使用说明
1. 直接在浏览器中打开 `index.html` 即可访问网站
2. 所有功能均为前端实现，无需服务器
3. 主题选择会自动保存，下次访问时会自动应用
