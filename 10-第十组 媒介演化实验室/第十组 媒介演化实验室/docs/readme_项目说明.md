媒介演化实验室

一、项目简介
媒介演化实验室是一个交互式教育网站，通过可视化时间轴和沉浸式模拟体验，展示人类媒介从口语时代到智能时代的演化历程。网站结合了前沿的Web技术与媒介研究理论，为用户提供了一场穿越媒介历史的数字旅程。

二、目标用户与应用场景
（一）目标用户
教育工作者：媒介研究、传播学、历史学等课程的辅助教学工具
学生群体：学习媒介史、传播理论的学生
研究学者：媒介演化研究的可视化展示平台
科技爱好者：对媒介技术发展感兴趣的人群
（二）应用场景
·课堂教学与展示
·媒介研究实验模拟
·学生自主学习
·博物馆/科技馆数字展览
·企业技术培训

三、网站信息结构
1.导航系统
首页：动态粒子背景与打字机效果标题
时间轴：六大媒介时代全景展示
项目：数据分析与智能推荐系统
关于：开发团队介绍

2.时间轴板块（六大媒介时代）
（1）口语时代 🔊
对话模拟器：角色对话创建与语音朗读
语音合成：支持暂停/继续功能
即时互动：实时对话显示
（2）文字时代 📝
数字书写板：在线画布书写体验
多种工具：画笔、橡皮擦、颜色选择
保存功能：导出书写作品
（3）印刷时代 📰
活字印刷游戏：汉字拼图互动游戏
打字机模拟：汉字与拼音对应显示
计时挑战：完成时间记录
（4）电子时代 📺
老式电视模拟：复古电视界面
收音机体验：虚拟电台切换
物理旋钮交互：模拟真实操作感
（5）网络时代 🌐
信息茧房模拟器：个性化信息流生成
兴趣标签筛选：6大兴趣类别
内容过滤演示：算法推荐原理展示
（6）智能时代 🤖
AI对话助手：DeepSeek API集成
智能问答：媒介历史相关问题
上下文记忆：连续对话功能

3.项目板块
媒介数据分析：三种图表展示媒介使用趋势
智能推荐系统：用户兴趣分析与内容推荐

四、主要交互功能
1.动态交互系统
粒子背景交互：鼠标悬停、点击产生粒子效果
平滑滚动导航：锚点跳转动画
模态框系统：弹出式内容展示，保持页面上下文
2.模拟器系统
口语对话模拟：双角色对话创建与语音合成
印刷拼图游戏：拖拽式汉字排序
数字书写体验：基于Canvas的绘画系统
复古设备模拟：电视、收音机虚拟操作
3.数据可视化
折线图：媒介使用趋势分析
柱状图：用户年龄分布
雷达图：用户兴趣维度
饼图/环形图：各类数据占比
4.智能系统
AI聊天助手：基于DeepSeek API的智能问答
信息推荐算法：兴趣标签匹配的内容生成
个性化体验：用户选择驱动的信息展示

五、Prompt 使用情况简述
本项目中我们基本上是全面采用了“Prompt驱动开发”模式，通过先规划整体框架，再细化到每一个功能的交互实现，以及相应的css样式设计，逐层的创建网页的每一个部分，最后再对每一个部分迭代，完善整个网页。另外我们将prompt生成图片替换为自行搜索的符合相应内容的图片，并将这些资源放在assets文件夹中。

六、网站运行与测试说明
可以通过直接打开“index.html”打开本网页。

功能测试
1. 导航栏
    点击“首页”、“时间轴”、“项目”、“关于”，应能平滑滚动到对应位置。
2. 时间轴
    点击六个时代图标，应弹出模态框，显示对应时代的详细内容和模拟器。
    测试每个模态框的关闭按钮和外部点击关闭。
3. 口语时代
    在输入框中输入角色A和角色B的对话，点击“+”按钮，应能添加对话并显示在聊天区域。
    点击“朗读对话”，应能听到语音朗读，并可以暂停和继续。
4. 文字时代
    在画布上使用画笔绘画，测试颜色选择、橡皮擦、清空、撤销和保存功能。
5. 印刷时代
    点击“开始游戏”，活字会出现在下方，拖拽到上方纸张的对应位置。
    测试拖拽功能，完成拼图后应有完成提示。
    在打字机部分输入文字，点击“开始印刷”，应能看到逐字打印效果。
6. 电子时代
    点击电视旋钮，应能切换图片。
    点击收音机旋钮，应能切换电台（注意：音频文件需要存在，否则会报错）。
7. 网络时代
    选择兴趣标签，点击“生成信息流”，右侧应显示对应标签的内容。
8. 智能时代
    在输入框输入问题，点击发送，应能收到AI回复（需要API密钥有效）。
9. 项目板块
    点击“媒介数据分析”和“智能推荐系统”，应弹出模态框显示对应的图表。
10. 信息茧房模拟器
    在网络时代模态框中点击“体验信息茧房模拟器”，应弹出新的模态框。
    选择兴趣标签，生成信息流。
兼容性测试
    在不同浏览器（Chrome, Firefox, Safari, Edge）中测试网站功能。
    在不同设备上（桌面、平板、手机）测试响应式布局。
性能测试
    检查页面加载速度，特别是粒子动画和图表渲染。
    测试同时打开多个模态框，确保没有内存泄漏。
